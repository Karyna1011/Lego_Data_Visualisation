# 🧱 LEGO: Анатомія іграшкової імперії

Інтерактивний дашборд для дослідження датасету LEGO-наборів 1990–2023 років.  
Побудований на чистому D3.js без фреймворків.

---

## 📊 Візуалізації

| # | Назва | Опис |
|---|-------|------|
| 01 | **Зростання асортименту** | Area chart + stacked по темах. Hover-курсор, анотації ключових подій (Star Wars 1999, криза 2008, Фільм LEGO 2014) |
| 02 | **Ціна за деталь** | IQR dot-plot по темах. Сортування, виділення, whisker-вуса |
| 03 | **Ціна vs деталі** | Scatter plot з лінією тренду. Фільтр по темі і року |
| 04 | **Склад року** | Waffle chart — кожен квадратик ≈ 1% наборів. Діапазон 1990–2022 |
| 05 | **Вік і ціна** | Bar chart з кольоровою шкалою за віком |
| 06 | **Радар тем** | Spider chart — порівняння до 3 тем за 5 показниками |
| 07 | **Декади** | Heatmap 1990s–2020s. Клік → топ-5 наборів декади |
| 08 | **Мініфігурки** | Area + точки. Клік → топ набори за к-стю фігурок |
| 09 | **Зал слави** | 10 легендарніших наборів з картинками |

## 🚀 Запуск

### Локально через WebStorm
1. Відкрийте папку проекту у WebStorm
2. Правою кнопкою на `index.html` → **Open In** → **Browser**
3. Сайт відкриється на `http://localhost:63342/...`

### Конвертація CSV у JS (якщо потрібно оновити дані)
```bash
python3 -c "
import csv, json
with open('lego_sets.csv') as f:
    data = list(csv.DictReader(f))
with open('lego_data.js', 'w') as f:
    f.write('const LEGO_RAW = ' + json.dumps(data) + ';')
print('Готово!')
"
```

## 🛠 Технології

- **D3.js v7** — всі графіки і візуалізації
- **Vanilla JS** — без фреймворків
- **CSS Custom Properties** — темна тема, адаптивність
- Дані: [Kaggle — LEGO Dataset by Maggie Akarn](https://www.kaggle.com/datasets/maggieakarn/lego-dataset)

## 📌 Ключові знахідки

- Після 1999 року (ліцензія Star Wars) асортимент LEGO зріс у рази
- Ліцензовані теми коштують дорожче за деталь ніж базові
- Набори для дорослих (16+) мають значно вищу середню ціну
- City - найбільша тема за кількістю наборів усіх часів
